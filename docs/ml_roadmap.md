# Machine Learning Roadmap

## Initial ML Task

Classify running activities into meaningful workout types.

### Candidate Labels

- Easy
- Long
- Threshold / Tempo
- Intervals
- Race

### Possible Input Features

- Distance
- Moving Time
- Avg Pace
- Avg HR
- Max HR
- Aerobic TE
- Training Stress Score
- Avg Power
- Total Ascent
- Climbing Density
- Avg Run Cadence

### Initial Models

1. Decision tree
2. Logistic regression
3. Random forest

### Evaluation

- Accuracy
- Confusion matrix
- Precision and recall by workout type
- Manual review of incorrect classifications
